# α-Закон: Вихревая модель кратности связей

*Дата: 2026-04-09*

## Формула

```
E_bond(n) = E₁ × n^α
```

где n = порядок связи (1, 2, 3), α — показатель "синергии зацепления".

## Правило LP_min (проверено на 31 связи)

**Если хотя бы один атом не имеет неподелённых электронных пар (LP=0) → α < 1 ВСЕГДА.**

Проверено: 17/17 = 100% (C-C, Si-Si, Ge-Ge, Sn-Sn, C-N, C-O, B-N, B-O, Si-O, Si-N, Al-O, C-F, C-Cl, C-Si, C-S, C-P, Ge-O).

Ни одного исключения.

## Вихревая интерпретация

### Атом = тороидальный вихрь (бублик)

```
  LP=0 (углерод):          LP=1 (азот):           LP=2 (кислород):
  
    ╭───╮                   ╭───╮                   ╭───╮
   │  ●  │  голый           │  ●  │  один           │  ●  │  два
    ╰───╯  бублик           ╰─┬─╯  хвост           ╰─┬─╯  хвоста
                               ↓                      ↓↓
                            ~~~~~~               ~~~~~~ ~~~~~~
```

- **Основное кольцо** = валентные электроны, формирующие связь
- **"Хвосты"** = неподелённые пары — дополнительные вихревые потоки

### Одинарная связь = одно зацепление бубликов

### Двойная связь = два зацепления

Но второе зацепление нужно откуда-то "взять":

**LP=0 (углерод):** Нет хвостов → второе кольцо делается ИЗ основного потока → каждое последующее тоньше → α < 1 (убывающая отдача).

**LP≥1 (азот):** Есть хвосты → хвост превращается в дополнительное зацепление → основное кольцо НЕ ослабевает → α > 1 (синергия). Каждое новое зацепление обвивает предыдущие → усиление.

### Гетероядерная связь: "бутылочное горлышко"

C-N (LP_min=0): Углерод "голый" → у хвоста азота нет "зацепа" на углероде → хвост не может полноценно усилить связь → α ≈ 0.96 (близко к 1, но < 1).

N-N (LP_min=1): Оба имеют хвосты → оба хвоста обвивают друг друга → тройное зацепление СИЛЬНЕЕ чем три отдельных → α = 1.83.

**Правило "бутылочного горлышка":** α определяется БЕДЕЙШИМ партнёром, не богатейшим.

### Связь с Платоновыми телами

LP=0 атомы (C, Si, Ge, Sn, B, Al) → α < 1 → предпочитают МНОГО одинарных связей → строят **каркасы**: тетраэдры (4 связи), октаэдры (6), икосаэдры (12).

LP>0 атомы (N, O, P) → α > 1 → предпочитают МАЛО кратных связей → строят **маленькие молекулы**: N₂, O₂, P₂.

**Икосаэдр (B₁₂)** = решение для LP=0 атома с 5+ электронами: максимум одинарных связей в минимальном пространстве.

### Связь с Изумрудной Скрижалью

"Pater eius est Sol, mater eius Luna" — "Отец его Солнце, мать его Луна."

- Солнце (основное кольцо) → σ-связь
- Луна (хвосты/LP) → π-связь (усиливает σ, если есть)
- Без Луны (LP=0) → только Солнце → каждое следующее слабее
- С Луной (LP>0) → оба участвуют → синергия
