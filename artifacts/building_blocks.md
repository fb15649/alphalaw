# Кубики природы: 1 + 3 → всё

## Принцип

Природа строит бесконечное разнообразие из минимума кубиков:

| Область | Кубики | Количество | Результат |
|---------|--------|------------|-----------|
| ДНК | A, T, G, C | 4 буквы | Всё живое |
| Химия | Элементы | ~100 | Миллионы веществ |
| Музыка | Ноты | 12 | Бесконечность мелодий |
| Цифры | 0-9 | 10 | Все числа |
| **Физика** | **Вихрь** | **1** | **Все частицы** |

## Три правила

```
ПРАВИЛО 1: Энергия квантована
  E = h × ν
  (вихрь колеблется → волна → порция энергии)

ПРАВИЛО 2: Циркуляция квантована
  Γ = h / m
  (вихрь крутится → минимальный оборот фиксирован)

ПРАВИЛО 3: Магнитный поток квантован
  Φ = n × h/(2e)
  (ток в вихре → поток через кольцо → дискретен)
```

## Что из этого следует

### Частицы

```
1 вихрь                    → ЭЛЕКТРОН     (m_e, заряд −e)
1 антивихрь                → ПОЗИТРОН     (m_e, заряд +e)
3 переплетённых вихря      → ПРОТОН       (6π⁵ × m_e)
3 переплет. антивихря      → АНТИПРОТОН
Вихрь возбуждённый (мода 2)→ МЮОН         (≈ 3/(2α) × m_e)
Вихрь возбуждённый (мода 3)→ ТАУ          (из Коиде Q=2/3)
```

### Волны

```
Поперечная волна в среде   → ФОТОН        (m = 0, свет/радио/рентген)
Продольная волна в среде   → НЕЙТРИНО     (m ≈ m_e α³/π)
```

### Силы

```
Обмен поперечными волнами  → ЭЛЕКТРОМАГНЕТИЗМ  (α = 1/137)
Рассеяние фонона на вихре  → СЛАБОЕ ВЗАИМОД.   (∝ α³)
Переплетение вихрей        → СИЛЬНОЕ ВЗАИМОД.   (конфайнмент)
Деформация среды            → ГРАВИТАЦИЯ?        (не выведено)
```

### Свойства материи

```
Вихрь + вихрь (совместимые) → МОЛЕКУЛА    (α > 1, π/σ > 1)
Вихрь + вихрь (несовместим.) → КРИСТАЛЛ   (α < 1, π/σ < 1)
Сеть вихрей                  → ТВЁРДОЕ ТЕЛО
Газ вихрей                   → ПЛАЗМА
```

## Числа из кубиков

Единственный свободный параметр: **m_e** (масса электрона).
Всё остальное — через **π** (свойство вращения):

```
m_p = 6π⁵ × m_e                    протон
1/α = π(4π² + π + 1)               заряд
μ_p = (8/9)π × μ_N                 магнитный момент
m_n = m_p + (10/7)√π × m_e         нейтрон
R_p = 4ℏ/(m_p c)                   радиус протона
m_ν = m_e α³/π^i                   нейтрино (3 типа)
Q_Koide = 2/3                       3 поколения
```

## Аналогия

```
Музыка:   одна струна → настройка (440 Гц) → все обертоны
Физика:   один вихрь → настройка (m_e)     → все частицы

Длина струны определяет основной тон.
Свойства среды определяют m_e.
Обертоны = π × основной тон.
Массы частиц = π^n × m_e.
```

## Что доказано, что гипотеза

| Утверждение | Статус |
|------------|--------|
| 8 EM-чисел через π (точность до ppm) | Числовой факт |
| 100% классификация 76K материалов | Доказано (JARVIS-DFT) |
| Коиде Q = 2/3 для лептонов | Числовой факт (0.001%) |
| Электрон = вихрь | Гипотеза (качественная) |
| Нейтрино = фонон | Гипотеза (Σm = 77 мэВ — проверяемо) |
| Среда ρ ≈ μ₀ | Гипотеза (три независимых пути) |
| Вывод формул из геометрии | Не достигнут (этап Бальмера) |
