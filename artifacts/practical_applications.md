## Что у нас есть сейчас

### Подтверждённые результаты
1. **Формула min_bond → Hv** (R²=0.84, 321 материал, Spearman ρ=0.94)
2. **α-закон** (LP=0→α<1, LP=1→α>1, 8/8 элементов = 100%)
3. **Частотное отношение → тип связи** (ρ=0.91, 19 типов связей)
4. **Рецептурная книга** (16 рецептов полиэдральных комбинаций)
5. **5 типов примесных эффектов** (классификация "мутаций" в графе)
6. **Random Forest** на 4 признаках: R²=0.91 (10-fold CV, 321 материал)

### Чего НЕ хватает для продукта
- Автоматическое определение min_bond из формулы (сейчас вручную)
- Автоматический парсинг "формула → полиэдральный граф"
- Валидация на 500+ с Vickers (а не Mohs) данными
- API Materials Project для масштабирования
