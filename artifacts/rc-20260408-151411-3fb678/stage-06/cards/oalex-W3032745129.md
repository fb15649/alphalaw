# DICE: A Monte Carlo Code for Molecular Simulation Including the Configurational Bias Monte Carlo Method

## Cite_Key
cezar2020dice

## Problem
Simulating flexible molecules in solution using Monte Carlo methods requires efficient conformational sampling.

## Method
Enhancements to the Configurational Bias Monte Carlo (CBMC) methodology using the molecular fragments concept in the DICE code.

## Data
Simulations of n-octane, 1,2-dichloroethane, and boron subphthalocyanine.

## Metrics
Acceptance rates, conformational sampling efficiency.

## Findings
The CBMC method, with the new fragments implementation, is a very good method for conformational sampling of complex moderately sized molecules (up to 150 atoms) in solution.
