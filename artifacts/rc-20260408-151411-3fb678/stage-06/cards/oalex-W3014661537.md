# Exploratory factor analysis and principal component analysis in clinical studies: Which one should you use?

## Cite_Key
alavi2020exploratory

## Problem
Confusion in the appropriate application and interpretation of Exploratory Factor Analysis (EFA) versus Principal Component Analysis (PCA) in clinical studies.

## Method
Methodological comparison and guidance using conceptual examples ('hospitalization stress' vs. 'hospitalization-related stressors').

N/A

## Metrics
N/A

## Findings
PCA is a data reduction technique that explains maximum variance using linear combinations, suitable for formative structures. EFA is a structure identification process that uncovers latent constructs explaining covariance, suitable for reflective structures.
