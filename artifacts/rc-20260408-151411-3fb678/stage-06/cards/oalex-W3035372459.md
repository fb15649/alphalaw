# Thermal Vibration Correlation Function Formalism for Molecular Excited State Decay Rates

## Cite_Key
shuai2020thermal

## Problem
Theoretical prediction of light-emitting quantum efficiency and excited state decay rates for complex molecules is computationally challenging.

## Method
Development of the Thermal Vibration Correlation Function (TVCF) formalism, implemented in the MOMAP software package.

## Data
N/A

## Metrics
Computation scaling (N^3).

## Findings
The TVCF formalism transforms the conventional Fermi-Golden Rule into a time-dependent formalism, converting exponential scaling summation to an N^3 scaling matrix production algorithm. It allows vibration mode mixing and is promoting-mode free.
