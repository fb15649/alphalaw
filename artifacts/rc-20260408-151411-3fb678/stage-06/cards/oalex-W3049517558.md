# Orbital graph convolutional neural network for material property prediction

## Cite_Key
karamad2020orbital

## Problem
Existing crystal graph neural networks lack features representing atomic orbital interactions, which are crucial for determining material properties.

## Method
Proposed the Orbital Graph Convolutional Neural Network (OGCNN), which includes atomic orbital interaction features and an encoder-decoder network.

## Data
Broad range of crystalline materials datasets.

## Metrics
Prediction accuracy (MAE), comparison with CGCNN, MBTR, SOAP.

## Findings
OGCNN significantly outperforms the crystal graph convolutional neural network and other state-of-the-art descriptors for material property prediction.
