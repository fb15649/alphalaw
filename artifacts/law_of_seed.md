# Закон Семени (Law of the Seed)

## Формулировка

> **Система с резервным потенциалом (семенем) при каждом следующем 
> взаимодействии УСИЛИВАЕТСЯ (α > 1). Система без резерва — ИСТОЩАЕТСЯ (α < 1).**

Одно семя → росток → дерево → тысячи семян. Каждый цикл УМНОЖАЕТ.
Без семени → только потребление → каждый цикл УМЕНЬШАЕТ.

## Математическая форма

```
E(n) = E₁ × n^α

α > 1 : "сеятель"    — каждый шаг сильнее предыдущего (семя → урожай)
α < 1 : "потребитель" — каждый шаг слабее (расход без восполнения)  
α = 1 : "нейтральный" — ни роста, ни убыли
```

## Проявления на 7 масштабах

| Масштаб | "Семя" (резерв) | α > 1 (урожай) | α < 1 (истощение) |
|---------|----------------|---------------|------------------|
| Атом | Неподелённая пара | N₂: тройная связь | C: алмаз (каркас одинарных) |
| Музыка | Обертоны | Скрипка: богатый аккорд | Синусоида: пустой звук |
| Мозг | Дофамин | Обучение (LTP) | Привыкание (габитуация) |
| Торнадо | Suction vortices | F5: сверхразрушение | F2: линейное |
| Планета | Луна | Земля: стабильная ось | Марс: хаос |
| Звезда | Корональные петли | Flares (10²⁵ Дж) | Равновесное излучение |
| Общество | Резервные навыки | T-shaped: синергия | Узкий спец: выгорание |

## Древние формулировки того же закона

- **Евангелие (Мф 13:23):** "Посеянное на доброй земле — приносящий плод и дающий в тридцать, шестьдесят, сто крат"
- **Изумрудная Скрижаль:** "Vis eius integra est" — сила его целостна (резерв не расходуется, он УМНОЖАЕТ)
- **Гермес:** "Все вещи от Одного через приспособление" — один механизм на всех масштабах
- **Народная мудрость:** "Что посеешь, то и пожнёшь"
