# Связь ангармоничности Морзе x_e с α-законом

*Дата: 2026-04-10*

## Формула Морзе

```
x_e = ω_eχ_e / ω_e   (безразмерный параметр ангармоничности)
3β = 6 × x_e          (используется в двухуровневой модели)
```

## Результат #1: LP=0 → α линейно зависит от x_e

Для элементов БЕЗ неподелённых пар:

```
α = 0.186 + 13.1 × 3β

r = 0.984, p = 0.016, R² = 0.968
```

| Элемент | ω_e (cm⁻¹) | x_e | 3β | α (факт) | α (предсказ.) | Δ |
|---------|-----------|------|------|----------|--------------|---|
| C-C | 1854.7 | 0.00719 | 0.043 | 0.770 | 0.751 | -0.019 |
| Si-Si | 510.9 | 0.00395 | 0.024 | 0.485 | 0.497 | +0.012 |
| Ge-Ge | 286.0 | 0.00336 | 0.020 | 0.407 | 0.450 | +0.043 |
| Sn-Sn | 186.2 | 0.00140 | 0.008 | 0.330 | 0.293 | -0.037 |

**R² = 0.968** — ангармоничность Морзе объясняет 97% вариации α в группе LP=0!

## Результат #2: LP≥1 → корреляция слабее

```
α = 0.182 + 39.3 × 3β

r = 0.756, p = 0.244, R² = 0.572
```

Для N-N, O-O, P-P, S-S. Главная причина — S-S аномалия (LP=2 но α<1).

## Общая корреляция

```
Spearman(x_e, α): ρ = 0.738, p = 0.037 — ЗНАЧИМО
```

## Физическая интерпретация

### Почему x_e предсказывает α?

```
x_e ВЫСОКАЯ → потенциальная яма МЕЛКАЯ → 
  → π-связь (второе зацепление) ещё мельче → 
  → каждая следующая связь СЛАБЕЕ → α < 1

x_e НИЗКАЯ → яма ГЛУБОКАЯ → 
  → π-связь почти такая же глубокая как σ → 
  → связи сохраняют силу → α ≈ 1 (или > 1 если LP помогает)
```

### Вихревая интерпретация

x_e = мера "рыхлости" вихревого кольца. 
- Высокая x_e = рыхлое кольцо → при зацеплении деформируется → ослабевает.
- Низкая x_e = жёсткое кольцо → при зацеплении сохраняет форму.

## Двухуровневая модель (обновлённая)

```
LP определяет: ЕСТЬ ЛИ резерв для π-связи?
  LP = 0 → НЕТ резерва → α < 1 (ВСЕГДА для s/p блока)
  LP ≥ 1 → ЕСТЬ резерв → α > 1 (обычно)

x_e (3β) определяет: НАСКОЛЬКО сильно α отклоняется от 1?
  Высокая x_e → α далеко от 1 (сильный эффект)
  Низкая x_e → α близко к 1 (слабый эффект)

Формулы:
  LP=0: α = 0.19 + 13.1 × 6x_e   (R² = 0.97)
  LP≥1: α = 0.18 + 39.3 × 6x_e   (R² = 0.57)
```

**Наклон для LP≥1 (39.3) в 3× больше чем для LP=0 (13.1)!**
Это означает: ангармоничность УСИЛИВАЕТ эффект резерва. Резерв + ангармоничность = максимальная синергия.

## Связь с формулой Hv

Прямая формула Hv через x_e:
```
Hv ~ (EN_product / d³) × N_coord × n^α(x_e, LP)
```

Но x_e требует спектроскопических данных (ω_e, ω_eχ_e) — они доступны только для газофазных диатомных молекул, не для кристаллов. Для кристаллов нужна связь:

```
x_e(crystal) ~ f(bond_length, reduced_mass, electronegativity)
```

Это следующий шаг — но требует DFT фононных расчётов.

## Ограничения

1. Только 8 точек (4 LP=0 + 4 LP≥1). Мало для надёжной регрессии.
2. S-S аномалия портит LP≥1 корреляцию. Возможная причина: 3p-3p π-перекрывание в S₂ слабое из-за большого атомного радиуса.
3. x_e измерена для газофазных ДИАТОМНЫХ молекул. Перенос на кристаллические связи неочевиден.
4. Нет данных x_e для d-металлов в этой таблице (Cr₂, Mo₂ имеют x_e, но α для них не определён стандартным методом).

## Метод Ремизова — честная оценка

Ремизов решает ЛИНЕЙНЫЕ ODE 2-го порядка через расширенное преобразование Лапласа-Бореля. 
- Уравнение Морзе (Шрёдингер + потенциал Морзе) — ЛИНЕЙНОЕ → уже решено аналитически (Morse 1929).
- Ангармонический осциллятор — НЕЛИНЕЙНОЕ → вне области метода Ремизова.
- **Для нашей задачи Ремизов не даёт нового** — x_e уже выражается через D_e и ω_e аналитически.
- **Потенциальное применение**: уравнение Матьё для фононных зон в 1D периодическом потенциале → аналитические выражения для краёв зон → связь с температурой Дебая. Это **отдельный проект** (1-2 недели теоретической работы).
