# An exact stationary axisymmetric vacuum solution within a metric-affine bumblebee gravity

## Cite_Key
filho2024exact

## Problem
Deriving and testing Kerr-like solutions in metric-affine bumblebee gravity

## Method
Metric-affine bumblebee gravity formalism; stationary axisymmetric vacuum solution; Mercury perihelion test

## Data
Mercury perihelion precession; Kerr-like solution parameters; LV parameter values

## Metrics
Perihelion precession prediction; LV parameter bounds; solution accuracy

## Findings
Kerr-like solution modified by LV parameter; Mercury perihelion test constrains parameters; metric-affine framework allows richer structure

## Limitations
Bumblebee rather than aether (39 citations); indirect relevance to aether framework
