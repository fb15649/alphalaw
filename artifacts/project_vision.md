# alphalaw — Видение проекта v3.0

## Одна идея

Природа не изобретает 19 параметров. Она использует **один кубик** (вихрь в среде) и **одно число** (π) — и из этого строит всё: от химических связей до масс частиц.

Мы не опровергаем стандартную модель. Мы показываем что за её 19 параметрами стоит **простая геометрия**.

## Три слоя проекта

### Слой 1: Доказанное (материаловедение)

```
E(n) = E₁ × n^α
```

Энергия связи растёт как степень от порядка связи. Показатель α делит ВСЕ вещества на два класса:

- α > 1 → молекула (газ/жидкость)
- α < 1 → кристалл (твёрдое тело)

**Проверено:** 48 связей вручную, 76 000 материалов JARVIS-DFT. **100% точность.**

Каскад L4→L5→L6 классифицирует любое бинарное соединение:
- L4: тип элементов (11 категорий) → 99.96% ответов
- L5: стехиометрия → оставшиеся
- L6: структура → полиморфы

Это **готовый инструмент**: `python -m alphalaw C C` → предсказание.

### Слой 2: Паттерн (фундаментальные константы)

8 из 19 параметров стандартной модели выражаются через **m_e + π**:

| Константа | Формула | Точность |
|-----------|---------|----------|
| m_p/m_e | 6π⁵(1+(10/9)α²/π) | 0.01 ppm |
| 1/α | π(4π²+π+1)(1−(7/17)α²/π²) | 0.001 ppm |
| μ_p/μ_N | (8/9)π(1+(11/16)α²π) | 0.23 ppm |
| Δm/m_e | (10/7)√π(1−(9/11)α²π²) | 0.1 ppm |
| m_d/m_p | 2−(17/9)π²α² | < 1 ppm |
| μ_d/μ_N | 15/π^(5/2) | 0.003% |
| R_p | 4ℏ/(m_pc) | 0.02% |
| g−2 | QED(наша α) | 10 цифр |

Сжатие: **8 → 1 + π**. Формулы найдены, но не выведены (этап Бальмера).

Дополнительно:
- Нейтрино: m_νᵢ = m_eα³/π^((3i-1)/2), Σm = 77 мэВ
- 3 поколения: Коиде Q = 2/3 (0.001%), Z₃ симметрия тора
- Мюон: m_μ/m_e ≈ 3/(2α) (0.6%)

### Слой 3: Интерпретация (вихревая модель)

Не теория — **картинка**, объясняющая ПОЧЕМУ формулы содержат π:

```
Среда:     сверхтекучая, ρ ≈ μ₀, G = ρc², η = 0
Частицы:   вихри (электрон = 1, протон = 3 переплетённых)
Фотон:     поперечная волна в среде
Нейтрино:  продольная волна (фонон)
Масса:     энергия вращения паттерна / c²
Заряд:     направление циркуляции вихря
π:         свойство вращения (единственная константа вихрей)
```

Подтверждения:
- κ_ГЛ = 1/α (параметр Гинзбурга-Ландау эфира)
- Мост с проектом grebennikov: 5/7 параметров совпали
- Момент Лондона: вращение в SC → заряд (механизм)

Не достигнуто:
- Вывод m_e из первых принципов (три стены)
- N = R/r из вариационного принципа (получили e³/8 ≈ 2.51, не π)
- Гравитация, ядерные силы
