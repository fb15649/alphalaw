# Фаза 2: Расширение гипотезы — Подробный план

## Общая цель
Перейти от "один полиэдр → одно свойство" к "граф полиэдров с разными связками → полный набор свойств".

---

## ШАГ 1: Граф полиэдральных связей (20 материалов)

### 1.1. Что делаем
Для 20 хорошо изученных материалов составляем "полиэдральный граф":
- Вершины = типы координационных полиэдров (Td, Oh, Ih, D3h, куб...)
- Рёбра = тип связи между полиэдрами (edge/corner/face/vdW/ionic)
- Вес ребра = сила связки (1-10 шкала)

### 1.2. Материалы для анализа
- 5 "чистых" (один тип полиэдра): алмаз, NaCl, Al₂O₃, SrTiO₃, графит
- 5 "смешанных" (два типа): MgAl₂O₄ (шпинель), YAG, оливин, слюда, топаз
- 5 "сложных" (три+ типа): биотит, турмалин, полевой шпат, глина (каолинит), тальк
- 5 "аномальных": α-бор, C₆₀, квазикристалл, перовскит BaTiO₃, алмаз+графит (карбин)

### 1.3. Формат описания
Для каждого материала:
```
МАТЕРИАЛ: [название]
ГРАФ: [полиэдр_A] --[тип_связи/сила]--> [полиэдр_B] --[тип/сила]--> ...
САМОЕ СЛАБОЕ ЗВЕНО: [какая связка]
ПРЕДСКАЗАНИЕ: твёрдость определяется [слабым звеном / сильным / средним]
ФАКТ: Vickers = X GPa, Mohs = Y
MATCH: предсказание совпало? да/нет
```

### 1.4. Метрики успеха
- Корреляция предсказанной "прочности слабого звена" с реальной твёрдостью (ρ > 0.5)
- Правило "слабого звена" работает в >80% случаев

### 1.5. Что это нам даёт? ✅ ВЫПОЛНЕНО

**Результат:** ρ = 0.984, R² = 0.88 — прочность слабого звена предсказывает твёрдость с точностью 85% (17/20 в пределах 3×). Это ЛУЧШИЙ предиктор из всех протестированных за всё исследование.

**Ключевое открытие:** bond_range (разница max−min в графе) АНТИКОРРЕЛИРУЕТ с твёрдостью (ρ = −0.59, p = 0.006). Гетерогенные материалы (графит: range=8) всегда мягкие. Гомогенные (алмаз: range=0) — определяются уровнем прочности.

**Формула:** log₁₀(Hv) ≈ 0.30 × min_bond − 0.76 (R² = 0.88)

- Альтернативные направления помимо шага 2:
  - A: Если граф работает → можно строить БАЗУ ДАННЫХ полиэдральных графов для всех 230 пространственных групп
  - B: Если "слабое звено" не работает → возможно нужна "средневзвешенная" или нелинейная комбинация
  - C: Можно сразу применить к предсказанию НЕ-твёрдости: теплопроводность, пьезоэффект, оптические свойства

## ШАГ 2: Правило "слабого звена" — количественная проверка

### 2.1. Что делаем
Формализуем: Hv_material ≈ f(min_bond_strength) или Hv ≈ f(weighted_average)?

Тестируем 3 гипотезы:
- H1: Hv = min(прочности всех связок) — "цепь рвётся по слабому звену"
- H2: Hv = среднее(прочности) — "всё вносит вклад"
- H3: Hv = геометрическое среднее — "компромисс"

### 2.2. Данные
Берём 20 материалов из шага 1 + добавляем 30 из предыдущего датасета = 50 материалов с известными графами и Vickers.

### 2.3. Метрики
R² для каждой из 3 гипотез. Какая лучше?

### 2.4. Что это нам даёт? (заполнить после)
- [результат]
- Альтернативные направления:
  - A: Если H1 (min) работает → "дизайн от слабого звена": хочешь тверже — усиль самое слабое
  - B: Если H3 (geomean) → материал = "аккорд", все связки вносят вклад нелинейно
  - C: Можно перейти к предсказанию АНИЗОТРОПИИ (разная твёрдость по разным направлениям = разные "слабые звенья" по осям)

## ШАГ 3: Рецептурная книга — 50+ комбинаций

### 3.1. Что делаем
Систематический каталог ВСЕХ известных комбинаций {полиэдр_A + связка + полиэдр_B}:

Сколько типов полиэдров: ~8 (Td, Oh, Ih, D3h, D4h, куб, prism, дамбелл)
Сколько типов связок: ~6 (corner, edge, face, vdW, ionic, metallic)
Теоретических комбинаций: 8×6×8 = 384, но реально существует ~50-70

### 3.2. Для каждой комбинации
- Примеры (3+ материала)
- Средняя твёрдость
- Средняя теплопроводность
- Характерные свойства (магнетизм, пьезо, оптика)

### 3.3. Метрики
- Покрытие: сколько % всех известных минералов описывается нашими комбинациями
- Предсказательная сила: R² на holdout наборе

### 3.4. Что это нам даёт? (заполнить после)
- [результат]
- Альтернативные направления:
  - A: "Рецептурная книга" = ИНСТРУМЕНТ для инженеров-материаловедов
  - B: Поиск "пустых клеток" — комбинаций, которые теоретически возможны но ещё не синтезированы → кандидаты для новых материалов
  - C: Связь рецептуры с функциональными свойствами (не только механическими)

## ШАГ 5: Фазовые переходы как смена связки

### 5.1. Что делаем
Описываем фазовые переходы как изменение полиэдрального графа:
- Плавление: все связки → "жидкие" (потеря дальнего порядка, Ih доминирует)
- Стеклование: "замораживание" графа с Ih-преобладанием
- Мартенситный переход: перестройка связок (corner→edge или Oh→Ih→Oh)
- Пьезоэффект: поворот полиэдров при сохранении связок

### 5.2. Данные
5 известных фазовых переходов с хорошими структурными данными:
- Fe: bcc↔fcc (мартенсит)
- BaTiO₃: cubic↔tetragonal (пьезо)
- SiO₂: quartz↔cristobalite↔stishovite (давление)
- H₂O: лёд Ih→лёд III→лёд VII (давление)
- Металлическое стекло: жидкость→стекло (закалка)

### 5.3. Метрики
- Предсказана ли температура/давление перехода из изменения графа?
- Предсказан ли тип перехода (1-го или 2-го рода)?

### 5.4. Что это нам даёт? (заполнить после)
- [результат]
- Альтернативные направления:
  - A: Предсказание НОВЫХ фазовых переходов при экстремальных давлениях
  - B: "Дизайн переключаемых материалов" — материалы, меняющие свойства при малом воздействии
