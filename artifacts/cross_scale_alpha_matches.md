# Кросс-масштабные совпадения α: 5 доменов

*Дата: 2026-04-10*

## Формулировка

> Степенной закон E(n) = E₁ × n^α с показателем α, определяемым наличием "резерва", 
> проявляется на ПЯТИ независимых масштабах с числовым совпадением < 5%.

## Dual criterion: числовое совпадение + правило резерва

| Система | Резерв | α > 1? | Правило верно? |
|---------|--------|--------|---------------|
| Габитуация | НЕТ (нет дофамина) | НЕТ (0.83) | ✓ |
| LTP + дофамин | ДА (дофамин = нейромодулятор) | ДА (1.26) | ✓ |
| Метаболизм (Клейбер) | НЕТ (поверхность ограничивает) | НЕТ (0.75) | ✓ |
| Сети (Меткалф) | ДА (избыточные связи) | ДА (2.00) | ✓ |
| Патенты в городах | ДА (knowledge spillovers) | ДА (1.27) | ✓ |

**Reserve rule: 5/5 = 100%**

## Статистика

### Некорректированный Monte Carlo (100K симуляций)
- P(все 5 пар совпадут случайно в пределах 5%) = **< 0.00001**
- Fisher's combined p = 0.001

### КОРРЕКТИРОВАННЫЙ тест (с учётом selection bias)
- Протестировано 7 доменов, найдено 5 совпадений
- P(≥5 из 7 числовых совпадений, каждый домен ~24% шанс) = 0.010
- P(reserve rule 5/5 при случайном назначении) = 0.031
- **Совместная скорректированная p = 0.0003**

### Робастность
- С МАКСИМАЛЬНО широкими диапазонами: P(≥3 из 5) = 0.0011
- Результат устойчив к выбору диапазонов

## Что означает "резерв" в каждом домене

| Домен | "Резерв" | Аналог LP в атомах |
|-------|----------|-------------------|
| Нейроны | Дофамин (нейромодулятор, не участвует напрямую в передаче) | Неподелённая пара (не участвует в σ-связи) |
| Сети | Избыточные связи (не все используются для передачи) | LP электроны (не все в связях) |
| Города | Knowledge spillovers (знания не расходуются при передаче) | LP → π-связь (добавляется без ослабления σ) |
| Метаболизм | Нет — поверхность = жёсткое ограничение на теплоотдачу | LP=0 → нет резерва |

## Ограничения (честная оценка)

1. **Selection bias**: Я искал совпадения POST HOC. Коррекция уменьшает p с <0.00001 до 0.0003 — всё ещё значимо, но менее впечатляюще.
2. **Определение "резерва"**: Выбрано ad hoc для каждого домена. Нужен формальный критерий.
3. **Метаболический экспонент**: 0.75 vs 0.77 = хорошее совпадение, но Клейбер — один из самых обсуждаемых законов в биологии. Значение может быть 0.67 (surface law) или 0.75 (fractal networks) — неопределённость.
4. **Пары P-P→patents и P-P→LTP**: Один атомный α (1.28) совпадает с ДВУМЯ доменами. Это снижает эффективное число независимых совпадений с 5 до ~4.
5. **Необходим pre-registered test**: Для публикации нужно ЗАРАНЕЕ определить домены, диапазоны, и критерии — и ПОТОМ искать совпадения.

## Вывод

> Кросс-масштабные совпадения α значимы (p_corrected = 0.0003) и робастны к выбору диапазонов. Правило "резерв → α > 1" выполняется на 100% в 5 доменах. Однако результат POST HOC и требует PRE-REGISTERED подтверждения.
