# Фаза 4: Полный каталог гипотез

## Контекст
Наша модель объясняет 27% твёрдости из формулы. Но мы тестировали ТОЛЬКО твёрдость. 
Если теория "вихревой геометрии" верна, она должна предсказывать ВСЕ свойства, 
зависящие от связей. Каждое свойство — отдельный тест теории.

---

## ГИПОТЕЗА 1: T_melt ∝ min_bond
**Суть:** Температура плавления = разрыв самого слабого звена. 
T_melt должна коррелировать с нашими формульными признаками СИЛЬНЕЕ чем Hv.
**Почему:** T_melt — чисто энергетический параметр (kT_melt ~ E_bond), 
без шума от дислокаций/дефектов.
**Тест:** R² формульных признаков → T_melt vs → Hv на JARVIS.
**Предсказание:** R²(T_melt) > R²(Hv) = 0.27

## ГИПОТЕЗА 2: Band gap ∝ delta_EN × connectivity
**Суть:** Ширина запрещённой зоны определяется РАЗНОСТЬЮ частот вихрей 
(delta_EN = разность электроотрицательностей) и типом связности.
**Почему:** Band gap = энергия "расщепления" между связывающей и антисвязывающей 
орбиталями. Чем больше delta_EN (разница "нот") → шире gap. 
Но металлы (делокализованные связи) → gap=0.
**Тест:** Формульные признаки → band gap на JARVIS (есть для 76K).
**Предсказание:** delta_EN и frac_TM — главные предикторы gap.

## ГИПОТЕЗА 3: Теплопроводность κ ∝ coherence / complexity
**Суть:** Теплопроводность определяется "когерентностью" фононов.
Простые структуры (мало атомов в ячейке) → фононы когерентны → κ высокое.
Сложные → рассеяние → κ низкое.
**Почему:** Правило Slack: κ ∝ 1/∛N_atoms. Наша "когерентность" = 1/log(N).
**Тест:** Формульные признаки → κ. Данные могут быть в JARVIS/MP.
**Предсказание:** n_elements и complexity — главные предикторы κ.

## ГИПОТЕЗА 4: Формульные признаки предсказывают ТИП материала
**Суть:** Не конкретное число (Hv, T_melt), а КЛАСС: металл, полупроводник, 
изолятор, сверхпроводник. Это ПРОЩЕ — и может работать лучше.
**Почему:** Классификация допускает нелинейные границы. 
frac_TM > 0.5 → скорее металл. delta_EN > 2 → скорее изолятор.
**Тест:** Классификация по band gap: metal (gap=0), semiconductor (0<gap<3), insulator (gap>3).
**Предсказание:** Accuracy > 70% из формулы.

## ГИПОТЕЗА 5: T_melt/T_boil = универсальная константа по классам
**Суть:** Отношение T_melt/T_boil для материалов одного "рецепта" (одинаковая 
полиэдральная комбинация) должно быть примерно постоянным.
**Почему:** Плавление = разрыв min_bond. Кипение = разрыв всех связей. 
Если все связи одинаковые (bond_range=0), T_melt/T_boil → const.
**Предсказание:** Для чистых элементов T_melt/T_boil ≈ 0.5-0.7 (факт: 
это правило Трутона в другой форме).

## ГИПОТЕЗА 6: Коэффициент теплового расширения α_T ∝ 1/T_melt
**Суть:** Мягкий потенциал → большое расширение → низкое T_melt. Жёсткий → малое → высокое.
**Почему:** Правило Грюнайзена: α_T × T_melt ≈ const для простых материалов.
**Тест:** Проверить: наши формульные признаки → α_T?
**Предсказание:** Если min_bond → T_melt, то min_bond → 1/α_T тоже.

## ГИПОТЕЗА 7: Энергия формирования ΔH_f отражает "консонанс" аккорда
**Суть:** Стабильные соединения (отрицательная ΔH_f) — те, чьи атомы 
"звучат консонантно" (EN-ratio близок к простой дроби).
**Почему:** Энергия формирования = разница между "аккордом" (соединение) 
и "отдельными нотами" (элементы). Консонантный аккорд = выигрыш энергии.
**Тест:** Формульные признаки → ΔH_f на JARVIS (есть для 76K!).
**Предсказание:** EN_product и r_ratio — главные предикторы ΔH_f.

## ГИПОТЕЗА 8: Объёмный модуль B ∝ EN_product / d_bond³
**Суть:** B (сопротивление сжатию) определяется плотностью энергии связей.
Плотность ∝ (сила связи) / (объём) ∝ EN₁×EN₂ / d³.
**Почему:** Из физики: B = -V × dP/dV. Для ионного кристалла B ∝ q²/(4·d⁴).
Наш EN_product — прокси для q². Наш d_bond — прокси для d.
**Тест:** Формульные признаки → B на JARVIS (есть для 25K).
**Предсказание:** R² > 0.3 из формулы.
