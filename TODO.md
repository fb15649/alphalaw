## Вихревая гипотеза (Empty Gap)

Идея: атомы как пульсирующие вихри с характерной частотой.
- Молекулы (α > 1): вихри синхронизируются (frequency locking)
- Кристаллы (α < 1): вихри конкурируют (diminishing returns)
- Gap [1.63, 1.97] ≈ [φ, 2]: "запретная зона" резонанса
- Аналогия с орбитами планет: Titius-Bode, Kirkwood gaps, числа Фибоначчи
- Исследовать: связь частоты вихрей с ионизационной энергией, колебательными модами, орбитальными частотами

### Маппинг: вихревая модель ↔ современная физика

| Вихревая модель | Современная физика | Формула |
|---|---|---|
| Частота пульсации вихря | Орбитальная частота электрона | ω = E_n / ℏ |
| Синхронизация вихрей | Резонанс орбиталей (bonding) | ψ_bond = ψ_A + ψ_B |
| Антисинхронизация | Антирезонанс (antibonding) | ψ_anti = ψ_A − ψ_B |
| Частота биений | Разница энергий σ и π связей | Δω = (E_σ − E_π) / ℏ |
| σ-связь и π-связь | Два "мода колебаний" одного вихря | — |

Gap [φ, 2] в терминах мод:
- π-мода сильная (Period 2, маленькие атомы) → R > 2 → молекула
- π-мода слабая (Period 3+, большие атомы) → R < φ → кристалл
- Между φ и 2 — нет стабильного режима

### Почему именно φ (золотое сечение)?

φ появляется когда система максимально далека от любого резонанса:
- KAM-теорема (динамические системы): самые устойчивые тори — с частотами в отношении φ
- Филлотаксис (ботаника): листья растут под углом 137.5° = 360°/φ²
- Квазикристаллы (Шехтман, Нобель 2011): икосаэдрическая симметрия ↔ φ

Гипотеза: R = φ — порог, ниже которого π/σ-отношение "слишком иррационально"
для кооперативного усиления. Связи не могут синхронизироваться → кристалл.

Проверка:
1. Посчитать π/σ overlap ratio для всех 37 bonds → коррелирует ли с φ?
2. Орбитальные частоты (ω = E_ionization / ℏ) Period 2 vs 3 → где отношение проходит через φ?
3. [ПРОВЕРЕНО] R_boundary ≠ φ. Gap = [2.016, 2.081] — вокруг октавы (2:1), не φ.
   НО: φ — точка уплотнения (6 bonds при R₂ ≈ φ±0.04). Аттрактор, не граница.
   Gap в paper [1.63, 1.97] был на ДРУГОМ определении R. На R₂=E₂/E₁ gap при R≈2.
   Это ЛУЧШЕ для модели: граница = октава = простейший резонанс 2:1.
   См. scripts/check_phi_boundary_v2.py
4. Колебательные частоты молекул (IR/Raman) → корреляция с α?

### Механизм: сохранение момента импульса ("эффект фигуриста")

Ключевое уравнение: L = I · ω = const
- Руки к телу (r↓) → момент инерции I↓ → частота ω↑
- Руки в стороны (r↑) → I↑ → ω↓

Температурная динамика вихрей:
- T↓ → r↓ → ω↑ (phonon hardening) → связи крепнут → сжатие
- T↑ → r↑ → ω↓ (phonon softening, 5-15%) → связи слабеют → расширение
- Механизм расширения/сжатия: именно L=const, а не "энергия → движение"

Подтверждения в физике:
1. Электронные орбитали: L = ℏ√(l(l+1)), сжатие орбитали (больше Z) → выше ω → выше E_ionization
   (F электроотрицательнее I именно поэтому)
2. Сверхпроводимость: T↓ → куперовские пары "стягиваются" → когерентный вихрь → R=0
3. Нейтронные звёзды: r от 10⁶ км до 10 км → ω от 1 об/месяц до 700 об/сек

### Нулевая энергия и "эфир Менделеева"

Вихрь не может замереть — и квантовая механика это подтверждает:
- Принцип неопределённости: Δx · Δp ≥ ℏ/2 → полная остановка невозможна
- При T=0 K остаётся нулевая энергия ZPE = ℏω/2
- He-4 при 0 K не замерзает (сверхтекучесть) — ZPE > межатомные силы
- 0 K — асимптота, не достижимое состояние (3-й закон термодинамики)

Исторический контекст:
- Менделеев зарезервировал "нулевой элемент" (Ньютоний/Короний) — среда легче водорода
- "Короний" оказался Fe¹³⁺, но интуиция о "чём-то перед водородом" перекликается с:
  квантовый вакуум (не пустой), поле Хиггса, нулевая энергия вакуума

В вихревой модели: 0 K — не "нет движения", а минимальная частота вихря (основная гармоника).
Вихрь пульсирует всегда. При T→0 выходит на фундаментальную частоту.

### Связь температуры с α-law

Предсказание 1 — α vs T_melt/T_dissociation:
- α > 1 (молекулы): π-моды гаснут первыми при нагреве → T_dissociation низкая (сотни K)
- α < 1 (кристаллы): σ-каркас из многих слабых связей → T_melt высокая (тысячи K)
- Проверяемо: scatter plot α vs T для 37 bonds (данные из CRC Handbook)

Предсказание 2 — α(T) имеет максимум:
- T слишком низкая: вихри "поджались", overlap между ними мал → α падает
- T оптимальная: overlap максимален, когерентность ещё есть → α максимален
- T слишком высокая: вихри расплылись, когерентность потеряна → α падает
- Аналогия: фигурист — слишком сжался (не дотянуться до партнёра),
  слишком раскинулся (теряет контроль)

Предсказание 3 — α стабилизируется при T→0:
- Вихрь выходит на основную гармонику → α перестаёт меняться

Где вихревая модель предсказывает лучше стандартной:
1. Ангармонический сдвиг частот — естественное следствие L=const + r(T)
2. Отрицательное тепловое расширение (NTE: ZrW₂O₈, ScF₃) — поперечные моды стягивают узлы
3. Фазовые переходы: soft mode theory (ω→0) = "вихрь замирает" → перестройка структуры

### Аналогия: интроверсия / экстраверсия

α — мера концентрации энергетического бюджета ("сила слабых связей" Грановеттера).

| Свойство         | α > 1 (молекула)           | α < 1 (кристалл)             |
|------------------|----------------------------|-------------------------------|
| Тип              | Интроверт                  | Экстраверт                    |
| Связи            | Мало, но сильные           | Много, но слабые              |
| Самодостаточность| N₂ летает один             | Si без 4 соседей не существует|
| T_melt           | Низкая (легко "уходит")    | Высокая (сеть держит)         |
| Координац. число | 1-2                        | 4-8-12                        |
| Аналогия         | 5 close friends            | 150 acquaintances (Данбар)    |

Предсказание: α обратно коррелирует с координационным числом (CN).

### Результат: α vs CN (проверено 2026-04-10)

Скрипт: scripts/alpha_vs_coordination.py, 31 bond с данными.
- Pearson r = -0.383, Spearman ρ = -0.443
- Средний CN: молекулы 1.9, кристаллы 4.3 (разница 2.3×)
- По бинам:
  - α > 1.5 (сильные молекулы): CN = [1, 1, 1, 2], mean 1.2
  - 1.0 < α ≤ 1.5 (слабые молекулы): CN = [2, 2, 3, 3], mean 2.5
  - 0.7 < α ≤ 1.0 (слабые кристаллы): CN = [1..12], mean 5.0
  - α ≤ 0.7 (сильные кристаллы): CN = [2..8], mean 4.0
- КЛЮЧЕВОЕ: α > 1.5 → CN всегда 1-2 (без исключений!)
- Корреляция не идеальная т.к. CN кристалла определяется геометрией, а не только α
- Однако тренд чёткий: интроверты (мало связей) = молекулы, экстраверты (много) = кристаллы

### Два плана: внутренний (α) и внешний (геометрия)

α — свойство внутреннего плана (энергия, содержание, "характер").
CN / space group — свойство внешнего плана (форма, структура, "дом").

Внутреннее определяет внешнее, но не однозначно:
- C: α=0.81 (кристалл) → алмаз CN=4, графит CN=3, фуллерен CN=3
- BN: α=0.71 (кристалл) → гексагональный CN=3, кубический CN=4
- Один характер → разные дома = полиморфизм

Каскад L4→L5→L6 = каскад от внутреннего к внешнему:
- L4 (тип элементов) = внутренняя природа (99.96% ответов)
- L5 (стехиометрия) = распределение внутренней природы
- L6 (структура) = внешняя геометрическая форма
L5/L6 нужны только когда внутренняя природа допускает несколько внешних форм.

α предсказывает на 100% потому что это свойство внутреннего плана.
CN коррелирует умеренно потому что внешний план имеет собственные ограничения
(дискретный набор: CN = 1,2,3,4,6,8,12 — геометрия упаковки сфер).

Связь с Цветком Жизни: допустимые CN — это "точки пересечения" в геометрическом
шаблоне пространства. Атом с данным α выбирает из дискретного набора.
CN=5 и CN=7 запрещены кристаллографией — кроме квазикристаллов (и там φ!)

Подтверждение: 199 полиморфных формул из Level 6 — одинаковый α, разный CN/плотность.

### Два пути: внутреннее→внешнее и внешнее→внутреннее

Путь 1 (α > 1, сильная личность): внутренняя природа создаёт структуру.
  N₂ навязывает миру свою форму. 1 полиморф. Чтобы сломать — 100+ ГПа.

Путь 2 (α < 1, слабая личность): структура/среда воспитывает.
  Si послушно принимает любую форму (diamond, β-Sn, amorphous, epitaxial).
  12+ полиморфов. Восприимчив к внешнему давлению.

Предсказание: число полиморфов обратно коррелирует с α.
  α > 1.5 → 1 форма (N₂, O₂)
  α ~ 0.5 → 10+ форм (Si, SiO₂)
  α ~ 0.7 → 3-5 форм (C, BN)
  S (α=0.68) → ~30 полиморфов (рекорд)

Практическое следствие: эпитаксия (внешнее→внутреннее) работает
только с низко-α материалами. Высоко-α "улетят" с подложки.

Оба пути реальны. α определяет какой путь доминирует.

### Полутона: α ≈ 1 — зона амбивертов

Три режима:
- α > 1.5: внутреннее побеждает → чистая молекула (газ)
- α ≈ 1.0: паритет → промежуточные состояния (полимер, стекло, жидкий кристалл, MOF)
- α < 0.5: внешнее побеждает → чистый кристалл

Пограничные случаи (α = 0.9–1.1):
- C-N (0.986): HCN газ, но полимеризуется в полиHCN
- S-O (0.978): SO₂ газ, но SO₃ самопроизвольно полимеризуется
- C-O (1.018): CO₂ газ, но при 40 ГПа → кристалл CO₂-V (как кварц!)
- C-S (1.075): CS₂ жидкость — между газом и твёрдым

Полутона в молекулах:
- Резонансные структуры: бензол C₆H₆ — bond order 1.5 (буквально полутон)
- Металлоцены: Fe(C₅H₅)₂ — связь Fe-C размыта по кольцу
- Радикалы: NO — неспаренный электрон ("незавершённая личность")

Связь с Empty Gap [φ, 2]:
Gap = зона неустойчивого равновесия. R в этой зоне означал бы "вечный амбиверт" —
ни стабильная молекула, ни стабильный кристалл. Природа запрещает такие R.

Устойчивость полутонов:
- В молекулах полутона менее устойчивы (стремление к определённости)
- В кристаллах полутона стабильнее (стёкла существуют миллионы лет)

### Платоновы тела и координационные числа

Разрешённые CN в кристаллах = вершины Платоновых тел:
- Тетраэдр (4 вершины) → CN=4: Si, SiO₂, алмаз
- Октаэдр (6 вершин) → CN=6: TiO₂, Al₂O₃, NaCl
- Куб (8 вершин) → CN=8 (ОЦК): Cr, Mo, W
- Икосаэдр (12 вершин) → CN=12 (ГПУ/ГЦК): Re, Cu, Au
- Додекаэдр (20 вершин) → CN=20: НЕ реализуется в обычных кристаллах

Ключевое разделение:
- Тела БЕЗ φ (тетраэдр, куб, октаэдр) → обычные кристаллы (периодические)
- Тела С φ (додекаэдр, икосаэдр) → квазикристаллы (апериодические, Шехтман 2011)

φ соединяет оба плана:
- Внутренний: граница Empty Gap при R = φ
- Внешний: φ встроено в геометрию додекаэдра/икосаэдра
- Квазикристаллы — материалы НА ГРАНИЦЕ: реализуют "запрещённую" 5-кратную симметрию

### Звёздчатые тела Кеплера-Пуансо

4 правильных невыпуклых многогранника — все содержат φ, все связаны с 5-кратной симметрией.
Гипотеза: если Платоновы тела = CN в кристаллах, то звёздчатые тела = квазикристаллические
координации. "Скрытые формы" внутри обычных тел.

### Музыкальная аналогия Empty Gap

α = отношение частот мод (π/σ). Gap = зона диссонанса.
- 2:1 (октава) = идеальный консонанс → молекула (R ≥ 1.97)
- φ:1 = максимальный диссонанс → граница кристалла (R ≤ 1.63)
- Между φ и 2: ни один устойчивый музыкальный интервал → пустая зона

Октаэдр (CN=6) → малая секста 8:5 = 1.60 ≈ φ — ближе всего к границе.
Октаэдрические кристаллы (TiO₂, Al₂O₃) на границе ионного/ковалентного = "полутон".
