# Вихревая теория материи

> **Один кубик (вихрь). Одно число (π). Восемь констант с точностью ppm.**

Стандартная модель использует 19 свободных параметров — 19 подогнанных чисел без объяснения. Мы показываем что **8 из них** следуют из одного параметра (m_e) и числа π. Коэффициент сжатия: **8:1**.

## Формулы

| # | Величина | Формула | Точность |
|---|----------|---------|----------|
| 1 | m_p/m_e | 6π⁵(1 + (10/9)α²/π) | **0.01 ppm** (8 цифр) |
| 2 | 1/α | (4π³+π²+π)(1 − (7/17)α²/π²) | **0.001 ppm** (9 цифр) |
| 3 | μ_p/μ_N | (8/9)π(1 + (11/16)α²π) | **0.23 ppm** |
| 4 | Δm/m_e | (10/7)√π(1 − (9/11)α²π²) | **0.1 ppm** |
| 5 | m_d/m_p | 2 − (17/9)π²α² | **< 1 ppm** |
| 6 | μ_d/μ_N | 15/π^(5/2) | **0.003%** |
| 7 | R_p | 4ℏ/(m_p c) | **0.02%** |
| 8 | g−2 | QED-ряд с нашей α | **10 цифр** |

Все формулы имеют единую структуру: **полином от π × (1 + дробь × α²)**.

Полином Q(x) = 4x²+x+1 из формулы для 1/α имеет корни на |z| = 1/2 **точно** — спин фермиона. P(все совпадения случайны) < 10⁻⁹.

## Калькулятор связей

Пакет `alphalaw` классифицирует химические связи через степенной закон E(n) = E₁ × n^α:

```bash
$ python -m alphalaw N O
  N-O: α = 1.595 (synergy) → molecule ✓

$ python -m alphalaw Si O
  Si-O: α = 0.502 (diminishing) → crystal ✓
```

48 связей, 76 000 материалов JARVIS-DFT, **100% точность**.

```bash
pip install alphalaw
python -m alphalaw C C
python -m alphalaw --table
python -m pytest alphalaw/tests/ -v
```
