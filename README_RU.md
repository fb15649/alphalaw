# Один закон от кварков до кристаллов

> **E(n) = E₁ × n^α — одна формула, все масштабы материи.**

## Открытие

Один степенной закон описывает рост энергии с числом составляющих — от субатомных частиц до химических связей:

```
Ядерный:   электрон(1) → пион(2) → протон(3)    α = 10.2, β = −3.1
Химический: одинарная → двойная → тройная связь    α = 0.5 − 1.8
```

Одна формула. Два масштаба. Число кубиков (вихрей, связей) определяет всё.

## Массы частиц из числа вихрей

n = число вихрей (торов в сверхтекучей среде):

| n | Частица | Масса (МэВ) | Формула | Точность |
|---|---------|-------------|---------|----------|
| 1 | Электрон | 0.511 | m_e (входные) | — |
| 2 | Пион π± | 139.6 | m_e × 2^(α+β ln2) | точно (входные) |
| 3 | Протон | 938.3 | m_e × 3^(α+β ln3) | точно (входные) |
| 4 | D_s мезон | 1968 | **предсказание: 1959** | **0.5%** |
| 5 | Ξ_c барион | 2470 | **предсказание: 2437** | **1.3%** |

Закон устойчивости: **2 вихря = нестабильно** (все мезоны распадаются). **3 вихря = стабильно** (протон живёт > 10³⁴ лет). Принцип табуретки — от мебели до кварков.

## Восемь констант через π

8 EM-констант = один параметр (m_e) + π. Сжатие: **8:1**.

| Величина | Формула | Точность |
|----------|---------|----------|
| m_p/m_e | 6π⁵(1 + (10/9)α²/π) | **0.01 ppm** |
| 1/α | (4π³+π²+π)(1 − (7/17)α²/π²) | **0.001 ppm** |
| μ_p/μ_N | (8/9)π(1 + (11/16)α²π) | **0.23 ppm** |
| Δm/m_e | (10/7)√π(1 − (9/11)α²π²) | **0.1 ppm** |

P(все совпадения случайны) < 10⁻⁹. Полином Q(x) = 4x²+x+1 из формулы 1/α имеет корни на |z| = 1/2 — спин фермиона.
